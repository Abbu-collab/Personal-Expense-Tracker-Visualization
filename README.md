# 💰 Personal Expense Tracker with Data Visualization

A professional Python-based Personal Expense Tracker Dashboard built using Streamlit, Pandas, and Plotly for tracking, analyzing, and visualizing expenses.

This project helps users understand spending patterns, monitor budgets, analyze payment methods, and generate financial insights using interactive dashboards and charts.

---

# 📌 Project Overview

Managing personal expenses manually can be difficult and unorganized.

This project solves that problem by:
- Recording expenses
- Analyzing spending habits
- Generating visual reports
- Providing interactive dashboards
- Helping users manage budgets effectively

The application simulates a real-world finance tracking system using CSV files and synthetic expense data.

---

# 🎯 Problem Statement

# ⚙️ Installation Guide

## Step 1: Clone Repository

```bash
git clone YOUR_GITHUB_REPOSITORY_LINK
```

---

## Step 2: Open Project Folder

```bash
cd Personal-Expense-Tracker-Visualization
```

---

## Step 3: Create Virtual Environment

### Windows

```bash
python -m venv venv
```

Activate environment:

```bash
venv\Scripts\activate
```

---

### Mac/Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## Step 4: Install Required Libraries

```bash
pip install -r requirements.txt
```

---

# ▶️ How to Run Project

## Step 1: Create Dataset

```bash
python src/create_dataset.py
```

---

## Step 2: Run Streamlit Dashboard

```bash
streamlit run app.py
```
